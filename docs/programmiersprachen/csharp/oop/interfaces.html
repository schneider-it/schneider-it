Interfaces C#

Ein Interface ist die Schnittstelle zwischen der View und dem Controller. Dies eignet sich besonders bei großen Projektarbeiten.

Interfaces

Was ist ein Interface?

Ein Interface ist die Schnittstelle von einer Klasse. Es gibt also vor, welche Methoden und / oder Attribute in einer Klasse vorkommen müssen. Dies ist eine große Hilfe bei Projektarbeiten, wo mehrere Menschen an verschiedenen Klassen arbeiten, aber die gleiche Schnittstelle benötigen, damit am Ende alles funktioniert.
Syntax
interface <Interfacename> { }
Beispiel:
interface IPerson
{

}

Methoden

Wenn man eine Methode als Schnittstelle in einem Interface definieren will, muss man folgende Syntax verwenden:
void Sleep(int hours);

Properties

Was eher selten vorkommt, ist die Definition eines Properties in einem Interface. Trotzdem hier ein Beispiel:
string Name { get; set; }
